# File-Organizer-With-Simple-GUI

A simple GUI-Based file sorter, built with Tkinter.
All that the user needs to do is input a valid file directory from their computer and press the 'Sort' button

## Features:
- Sorts files by category (Images, Documents, Video, Audio, Presentations)
- Simple GUI interface
- Modular architecture

## Technologies
- Python
- Tkinter
- pathlib
- shutil
